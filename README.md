
# File Sorter in Rust

A simple file sorter in rust that organizes your files from a folder to (A-Z).
## Getting Started

### Prerequisites

- [Rust](https://www.rust-lang.org/learn/get-started) must be installed on the operating system.
- Cannot be a dumbass

### Installation

1. Clone the **repository**:
```bash
cargo build 
```
2. Build the **project**:
```bash
cargo build --release
```
3. Run the **project**: 
```bash
cargo run
```
